Compare & spot the bombiest format

Group by set or color, then switch metric to Avg IIH (how much a card overperforms when actually drawn) or R/M bomb rate (share of rares+mythics with a high GIH WR) to see which format or color runs hottest at the top end. Bars reflect the filters above. "Weighting" changes every metric here from a per-card average to a per-draw one — e.g. weighting by % GP or by # drawn (nGIH) makes a card that's actually played/drawn constantly count for more than one that's almost never in a deck, so a heavily-maindecked staple no longer counts the same as a niche sideboard card.

Common/Uncommon quadrant breakdown

For each set, "baseline" is the x-intercept of a Common+Uncommon-only regression of IIH on GIH WR — roughly that set's average GNS WR. Every card then falls into one of four quadrants: Strong & swingy (above baseline, IIH>0 — genuine impact cards), Deck-rider (above baseline, IIH<0 — looks good only because its decks are strong), Chaff (below baseline, IIH<0 — true filler), and Hidden gem (below baseline, IIH>0 — held back by bad company but pulls its weight when drawn). Baseline is fixed per set from the full Common+Uncommon pool — filtering by color re-slices which cards show, not where the line sits.

GIH WR quartiles by set

For the chosen rarity, each set's cards are split into 4 equal-sized quartiles by GIH WR — computed independently per set (so ATLA Mythic quartiles use only ATLA Mythic cards, never mixed with other rarities or sets). Bars show the mean GIH WR within each quartile with ±1 SEM error bars. Quartile cutpoints are fixed per set from the full color range; the Color filter above re-slices which cards populate each bar, not where the cutpoints sit. Source: 17lands-style card-ratings exports for each set. GIH WR = win rate in games where the card was drawn; GNS WR = win rate in games where it was seen but not drawn; IIH = GIH WR minus GNS WR; ALSA = average last seen at (lower = picked earlier). Blank cells mean too few recorded games for a reliable rate. Colorless = no color letters; multicolor cards keep their exact color combo in the Color column but are filed under one Multicolor filter. "R/M bomb rate" and "R/M − C/U gap" compare Rare+Mythic cards against Common+Uncommon cards within each bar and ignore the Rarity filter above (they need both tiers present); the bomb-rate threshold (GIH WR ≥ 58%) is a fixed, simple cutoff, not a statistically derived one. "% cards with IIH > 0 (floor rate)" respects the Rarity filter — group by Set with Rarity set to Common (or Uncommon) to compare how often a set's commons/uncommons are worth being happy to draw, rather than just deckbuilding filler. % GP = the share of games a card was in the player's pool that it actually got maindecked, a direct play-rate signal; "Min % GP played" is a hard exclusion applied everywhere (including quadrant/quartile baselines), while "Weighting" in the Compare panel is a softer re-weighting that keeps every card but counts frequently-drawn ones for more.
